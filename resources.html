Publications Resources

Code

  • treetools: A growing collection of algorithms for treebank tree processing.
  • evalb-lcfrs: An extension of evalb bracket scoring for PLCFRS parses.
  • uparse: Discontinuous shift-reduce parsing.
  • rparse: Discontinuous data-driven parsing with PLCFRS
  • TuLiPA: A parsing environment for variants of tree-adjoining grammar.
  • More on github.

Resources

  • Discosuite: A test suite for German discontinuous constituency structures, based on the TIGER treebank. More information. Download (Version 1.0, .csv).
  • Penn Treebank coordination annotation: An annotation layer for the Penn Treebank which marks coordinating punctuation. Available through the LDC.
  • Twotiger: A conversion script which reduces the block degree of the TIGER treebank to 2. Download (.tar.gz).
